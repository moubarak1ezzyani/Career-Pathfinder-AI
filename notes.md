# Notes
## Extract text from iamge
OCR (Reconnaissance Optique de Caractères) : La librairie la plus connue et gratuite pour faire cela est pytesseract (qui utilise le moteur Tesseract de Google), ou encore easyocr (très performante).

## load and read a PDF file
* Use pypdf for basic text extraction and PDF manipulation. 
* Use pdfplumber for detailed layout and table analysis.
* Use PDFQuery for targeted data extraction using selectors. 
* Use IronPDF for advanced features like rendering and editing.
